# Powerbi-student-feedback-dashboard
# Project Overview :- This project analyzes student feedback data collected from various academic and co-curricular events to evaluate overall satisfaction, sentiment, and performance across departments, event types, and demographics.
  An interactive Power BI dashboard was developed to help institutions understand student experiences, identify improvement areas, and support data-driven decision-making for academic and event planning.
# Objectives
  - Measure overall student satisfaction levels
  - Analyze average ratings by department, event type, and event name
  - Perform sentiment analysis on student feedback
  - Compare feedback across year of study and gender
  - Identify strengths and improvement opportunities in events
# Dataset Details
  - Source: Student feedback survey data
  - Data Type: Educational & Sentiment Analytics
# Key Fields:
  - Event Date
  - Event Type (Seminar, Workshop, Cultural)
  - Event Name
  - Department
  - Year of Study
  - Gender
  - Rating (1–5)
  - Sentiment Score
  - Feedback Comment
# Data Cleaning & Preparation
  - Removed duplicate and incomplete responses
  - Standardized categorical values (event types, departments)
  - Converted rating fields to numeric format
  - Created sentiment categories (Positive, Negative, Neutral)
  - Built calculated measures using DAX:
  - Average Rating
  - Sentiment Count
  - Rating Distribution
# Dashboard Features
  * Key KPIs
    - Average Rating
    - Total Sentiment Score
    - Sentiment Count
  * Visual Insights
    - Average Rating by Department
    - Average Rating by Event Type
    - Average Rating by Event Name
    - Sentiment Distribution (Positive, Negative, Neutral)
    - Overall Rating Distribution (Very Satisfied → Very Dissatisfied)
  * Interactive Filters
    - Event Date range
    - Year of Study (FY, SY, TY, Final Year)
    - Event Type
    - Department
    - Gender
# Tools & Technologies
  - Power BI – Dashboard creation & data visualization
  - DAX – Calculated measures & KPI
  - Excel / CSV – Data preprocessing

# Key Insights
  - Seminars received the highest average ratings
  - Certain departments consistently performed better in student satisfaction
  - Majority of feedback showed a positive sentiment
  - Workshops had balanced feedback with scope for improvement
  - Cultural events showed varied satisfaction levels
# Business / Academic Value
  - Helps institutions improve event planning and quality
  - Identifies departments needing attention
  - Enhances student engagement and satisfaction
  - supports evidence-based academic decisions
